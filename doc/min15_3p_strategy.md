# Min15 3% 추세추종 전략

## 개요

15분봉 기반의 추세추종 전략으로, 명확한 추세 구간에서만 진입하고
부분 익절 + 트레일링 스탑으로 수익을 극대화한다.

## 핵심 아이디어

> "추세를 따라 진입하되, 수익은 끝까지 끌고 간다"

- 추세가 확인되면 눌림목(롱) 또는 되돌림(숏)에서 진입
- 수익 +3% 도달 시 절반만 확정, 나머지는 트레일링으로 추세 끝까지 보유
- 손절은 변동성(ATR)에 맞춰 동적으로 설정

## 사용 지표

| 지표 | 설정 | 용도 |
|------|------|------|
| EMA 20 | 15분봉 | 단기 추세 방향 |
| EMA 120 | 15분봉 | 장기 추세 방향 |
| RSI 14 | 15분봉 | 과열/과냉 필터 |
| ATR 14 | 15분봉 | 변동성 측정 (손절폭 결정) |

## 추세 판단

**상승 추세 (UPTREND):**
- 현재가 > EMA 20 AND 현재가 > EMA 120
- EMA 20 기울기 > 0 (최근 3봉 대비 상승 중)
- EMA 120 기울기 > 0

**하락 추세 (DOWNTREND):**
- 현재가 < EMA 20 AND 현재가 < EMA 120
- EMA 20 기울기 < 0
- EMA 120 기울기 < 0

4가지 조건이 **모두** 충족되어야 추세로 인정한다.
하나라도 불일치하면 "NONE" → 진입하지 않음.

## 진입 조건

| 방향 | 조건 |
|------|------|
| 롱 진입 | UPTREND + RSI < 60 (과열 아닌 구간에서 진입) |
| 숏 진입 | DOWNTREND + RSI > 40 (과냉 아닌 구간에서 진입) |

RSI 필터의 의미: 추세 방향은 맞지만 너무 과열된 상태에서는 진입하지 않는다.
RSI가 60 미만(롱) / 40 초과(숏)일 때만 진입하여 눌림목을 노린다.

## 포지션 관리 흐름

```
진입 (시장가)
│
├─ 초기 손절 설정: 진입가 ± ATR × 2.5
│   (변동성이 크면 SL 넓게, 작으면 타이트하게)
│
├─ [수익 방향] +3% 도달
│   ├─ 포지션 50% 시장가 청산 (수익 확정)
│   ├─ 손절을 진입가로 이동 (본전 보장)
│   └─ 나머지 50%는 트레일링 스탑 모드
│       └─ 최고점(롱)/최저점(숏) 대비 ATR × 2 후퇴 시 청산
│
├─ [손실 방향] -5% 도달
│   ├─ 잔고 충분 → 추가매수 (2배 물량)
│   │   └─ 새 평균가 기준 -7% 도달 시 전량 손절
│   └─ 잔고 부족 → 즉시 전량 손절
│
└─ [변동성 손절] ATR × 2.5 SL 도달
    └─ 전량 손절
```

## 수익 극대화 메커니즘

### 1. 부분 익절 (Half-Exit)
+3%에서 전량 청산하면 이후 +10%, +20% 추세를 놓친다.
50%만 확정하고 나머지를 남겨둠으로써 **큰 추세에서 대형 수익**을 포착한다.

### 2. 트레일링 스탑 (Trailing Stop)
남은 50%는 고정 TP 없이, 가격이 올라갈수록 SL이 따라 올라간다.
ATR × 2 폭의 되돌림이 오면 청산 → 추세의 대부분을 먹을 수 있다.

### 3. 본전 보장
부분 익절 후 SL을 진입가로 옮기므로, 나머지 50%는
**최악의 경우에도 손실 0원**. 부담 없이 추세를 끝까지 탈 수 있다.

### 4. ATR 동적 손절
고정 % 손절(예: -3%)은 변동성이 큰 장에서 너무 쉽게 터진다.
ATR 기반으로 "현재 시장이 평소에 얼마나 움직이는지"를 반영하여
불필요한 조기 손절을 방지한다.

## 추가매수 (Averaging Down)

-5% 하락 시 2배 물량으로 추가매수하여 평균 진입가를 낮춘다.
추가매수 후에는 새 평균가 기준 -7%에서 손절한다.
추가매수는 **1회만** 허용하여 과도한 위험 노출을 방지한다.

## 리스크 관리

| 항목 | 설정 |
|------|------|
| 레버리지 | 3배 |
| 1회 진입 금액 | 잔고의 10% |
| 최대 동시 포지션 | 심볼당 1개 |
| 추가매수 | 최대 1회 (2배 물량) |
| 최대 손실 (추가매수 전) | ATR × 2.5 (가변) |
| 최대 손실 (추가매수 후) | 평균가의 -7% |

잔고의 10%만 사용하므로 최대 10개 종목을 동시 운용 가능.
레버리지 3배 적용 시 실질 노출은 잔고의 30% (종목당).
